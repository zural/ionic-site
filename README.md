ionic-site
==========

Repo for the ionicframework.com site


gulp watch uses LiveReload. You may have to up your max file limit with the following command:

    ulimit -n 5000


## Local Build

    jekyll serve -w

    npm install
    
    sudo npm install -g gulp

    gulp watch

    http://localhost:4000/
