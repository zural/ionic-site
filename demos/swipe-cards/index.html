Swipeable Card UI

With the Ionic HTML5 mobile framework and AngularJS

See the Pen by Ionic (@ionic) on CodePen.

Swipeable cards are an emerging mobile UI, seen in new apps like Jelly.

Cards make it easy to display modal content that fills the center of the screen, letting the user swipe it away to bring up a new bit of content.

We built a similar UI that you can use in your own Ionic apps today! To use it, check out the Ionic Swipe Cards repo and follow the guide in the README.

Or, download the starter Cordova project below which contains everything you need to build and run your own card-based Ionic app!
